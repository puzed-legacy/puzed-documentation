Records

Edit this pageLast updated: 2020-11-22

What are records

Every collection holds onto records which are managed through the configuration you set when creating it originally.

Available Methods

Search through records

List all the records in a collection

URL: https://YOUR-DATABASE.bitabase.net/:collectionName
Method: GET
Inputs: Not Applicable
Outputs: Array of records
Filtering: ?query={} ?fields=["dateCreated"]
Pagination: ?limit=10 ?offset=0 
fetch('https://YOUR-DATABASE.bitabase.net/:collectionName', {
  method: 'get'
})

Example Response:

{
  "count": 1,
  "items": [{
    "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "name": "test",
  }]
}
